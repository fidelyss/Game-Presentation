# 🤖 Introduction

This project is a **recreation and study project** inspired by the award-winning website design from **Zentry**.

The goal of this project is to explore advanced **modern web UI/UX techniques**, including:

- scroll driven animations
- geometric transitions
- immersive storytelling with video
- smooth interactions

The implementation was developed using **React**, **GSAP**, and **TailwindCSS**, focusing on building a modern interface similar to websites commonly featured on **Awwwards**.

This project is part of my journey to improve my **front-end animation and UI engineering skills**.

---

# ⚙️ Tech Stack

This project was built using the following technologies:

- **React.js**
- **GSAP (GreenSock Animation Platform)**
- **Tailwind CSS**
- **Vite**

---

# 🔋 Features

### 🎬 Scroll Based Animations
Dynamic animations triggered as the user scrolls through the page.

### 🔷 Geometric Clip Path Transitions
Creative layout transitions using CSS `clip-path`.

### 🧊 3D Hover Effects
Interactive 3D transformations that respond to user interaction.

### 🎥 Video Storytelling
Background and section videos integrated into the storytelling of the page.

### ⚡ Smooth UI/UX
Focus on fluid motion, smooth transitions and polished interactions.

### 📱 Fully Responsive
Optimized for desktop, tablet and mobile devices.
